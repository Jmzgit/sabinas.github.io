Getting Started

Sujon. is a HTML/CSS template based on " Materialize " framework. It allows to built impressive, better & faster, mobile friendly websites.

HTML Structure

This is the base of each HTML file:

                <!-- ==================== header-section Start ==================== -->
                <header>
                ...
                </header>
                <!-- ==================== header-section End ==================== -->


                <!-- ==================== section Start ==================== -->
                <Section>
                ...
                </section>
                <!-- ==================== section End ==================== -->


                <div>
                ...
                </div>
                <!-- /.class-name /#ID-name-->


                <!-- ==================== footer-section Start ==================== -->
                <footer>
                ...
                </footer>
                <!-- ==================== footer-section End ==================== -->

Grid System

Sujon. Template supporst all features of Materialize Grid System - go to Materialize's page and check how does it work.
