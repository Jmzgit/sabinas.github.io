Slider

  • 1. Open the file "index2.html"
  • 2. Find the following code 
       
                            <div id="lionhero" class="owl-carousel owl-theme">
                                ...
                            <div/>
    this code holds the slider/carousel
  • 3. within the code above, there are divs with class name- 'item'. these divs hold codes for individual slides. Inside each div with 'item' class there is a div with class- 'slide one' / 'slide two' / 'slide three' etc. This class has the slide image set as background.
  • 4. to change the images go to 'style.css' file in css folder and find the following code- 
    
                            .slide.one {
                                background: url(../images/slider/slide1.jpg);
                            }

                            .slide.two {
                                background: url(../images/slider/slide2.jpg);
                            }

                            .slide.three {
                                background: url(../images/slider/slide4.jpg);
                            }
    now change the image refernce as you wish.
